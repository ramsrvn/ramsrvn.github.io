Sharwan Ram is a highly skilled Software Engineer 2 in Product Security at [Garmin International](https://www.garmin.com/), specializing in Static Analysis (SAST), Software Composition Analysis (SCA), Software Bill of Materials (SBOM), automation, and vulnerability management. He leads SBOM integration and adoption, developing automation scripts to streamline its implementation and enhance third-party risk management. Additionally, he maintains critical security libraries across enterprise products, ensuring compliance and proactive vulnerability prevention. He actively participates in EU Radio Equipment Directive (RED), Cyber Resilience Act (CRA), US Cyber Trust Mark, and Executive Order 14028 compliance discussions.

Previously, Sharwan led the enterprise-wide integration of Cppcheck to improve code quality and security. He also developed automation scripts and a Python package to optimize static analysis workflows and issue prioritization. He also mentored team members on product security processes, fostering a culture of security best practices and continuous improvement

In past, Sharwan worked as a Software Engineer at [EnsoftCorp](http://ensoftcorp.com/), where he optimized legacy systems and developed Eclipse plug-ins for software analysis. He has also served as a Software Developer and Security Analyst on the [DARPA's Space/Time Analysis for Cybersecurity (STAC)](http://www.darpa.mil/program/space-time-analysis-for-cybersecurity) program at Iowa State University, developing advanced program analyzers to detect sophisticated vulnerabilities, he was a top-performing team member.

Sharwan holds two master’s degrees: an M.S. from Iowa State University and an M.Tech from MNIT Jaipur. He received recognition from [MIT](https://www.mit.edu/) for his contributions to the [Kerberos](https://github.com/krb5/krb5). His research on automated documentation generation earned the IEEE Best Challenge Entry Award at DySDoc3.
